# PayFlow

PayFlow — сатып алушы құқығын қорғайтын цифрлық чек жүйесі.

## MVP (жұмыс істейтін нұсқа)
- Чек жасау
- Аты, номер, сома, тауар
- Төлем қосымшасын таңдау
- PDF чек
- QR-код
- Қолмен жазылатын ИПС
- PDF-ті жіберуге дайын формат

Бұл нұсқа:
- комиссияға
- көрмеге
- қорғауға
толық жеткілікті.
